Original PDC estimation

PDC analysis results are saved in c struct variable. See asymp_dtf.m or issue >> help asymp_pdc command for more detail.

nFreqs = 128;
metric = 'euc'; % Euclidean metric = original PDC definition
alpha = 0.01;
c = asymp_pdc(u,A,pf,nFreqs,metric,alpha);

c.Tragct = Tr_gct;         % Assigning GCT results to c struct variable.
c.pvaluesgct = pValue_gct;

$|PDC(\lambda)|^2 Matrix Layout Plotting

flgPrinting = [1 1 1 2 3 0 1]; % overriding default setting
flgColor = 0;
w_max=fs/2;

strTitle = ['5-dimension linear VAR[3] Model: [N=' int2str(nSegLength) ...
             'pts; IP=' int2str(c.p) ']'];  %'; '  datestr(now) ']'];
vTitleBar = 'Baccala & Sameshima (2001) Example 3';

[h1,~, ~] = xplot(vTitleBar,c,...
                          flgPrinting,fs,w_max,chLabels,flgColor);
xplot_title(alpha,metric,'pdc', strTitle);
